SimPE External Dependencies
===========================
Last updated: 2026-03-17


DLLs IN THIS FOLDER (lib/)
---------------------------
These are closed-source third-party libraries with no available source code.
They must be kept here and are referenced by projects via relative HintPath.

  (none — all previously required DLLs have been replaced or eliminated)


NUGET PACKAGES (auto-restored on build, no DLL copy needed)
------------------------------------------------------------
These are declared as <PackageReference> in the relevant .csproj files
and are downloaded automatically by dotnet restore / MSBuild.

  SharpZipLib (ICSharpCode.SharpZipLib) v1.4.2
    Purpose: ZIP/GZip/BZip2 compression — replaces the old ICSharpCode.SharpZipLib.dll

  SharpCompress v0.47.0
    Purpose: Additional archive format support — replaces the old 7z.dll

  BCnEncoder.Net v2.3.0
    Purpose: BCn (DXT/BC) texture compression and decompression

  Pfim v0.11.4
    Purpose: TGA and DDS image loading — replaces the old soil2.dll native dependency
    Used by: SimPE.Filehandlers (PictureWrapper), SimPE.RCOL (DDSTool)

  DockPanelSuite (WeifenLuo.WinFormsUI.Docking) v3.1.1
    Purpose: Document-area tab/docking framework — replaces SandDock.dll (x86-only, commercial)
    Used by: SimPE.Main, SimPE.WorkSpaceHelper

  System.Drawing.Common (via Windows SDK targeting)
    Purpose: GDI+ drawing — required for net8.0-windows projects


SOURCE PROJECTS (formerly closed-source DLLs, now decompiled and in-tree)
--------------------------------------------------------------------------
These DLLs have been decompiled and are now built from source as part of
the solution. The original DLL files are no longer used.

  ambertation.3D.dll        -> Ambertation.3D\
  ambertation.utilities.dll -> Ambertation.Utilities\
  ambertation.3D.gl.binding.dll (formerly ambertation.3D.mdx.binding.dll,
                             originally Managed DirectX)
                            -> Ambertation.3D.Gl.Binding\  (rewritten using OpenTK)
  NetDocks.dll              -> NetDocks\
  xsi.lib.dll               -> xsi.lib\
  AdvancedForms.dll         -> AdvancedForms\

  XPCommonControls.dll      -- ELIMINATED. All XPGradientPanel controls were
                               replaced with standard System.Windows.Forms.Panel.

  Eyefinder.dll             -- ELIMINATED. UI controls replaced in OptionForm.cs.
  Navisight.dll             -- ELIMINATED. UI controls replaced in OptionForm.cs.

  SandDock.dll              -- REPLACED by NuGet DockPanelSuite (WeifenLuo.WinFormsUI.Docking).
                               Document tabs now use WeifenLuo DockPanel/DockContent.
                               Internal plugin tabs (HGBH, OWDockForm) use System.Windows.Forms.TabControl.
                               This was the last x86-only blocker; SimPE can now target AnyCPU/64-bit.

  soil2.dll                 -- REPLACED by NuGet Pfim (TGA/DDS) + BCnEncoder.Net (DXT).
                               No native DLL required.

  ICSharpCode.SharpZipLib.dll -- REPLACED by NuGet SharpZipLib (see above)
  7z.dll                      -- REPLACED by NuGet SharpCompress (see above)
